# Changelog

## [2.1.1] - 2026-07-20

### Fixed

- 修复 WebView2 返回非空 `scrollIntoView` 结果时，React 将其误作 effect 清理函数并导致工作室页面崩溃的问题。
- 新增覆盖 WebView2 返回值差异和组件卸载流程的回归测试。

## [2.1.0] - 2026-07-20

### Added

- 双入口创作流程，以及定位主题、世界规则、文风边界、人物关系、剧情、分卷、章节和场景的独立 Agent 审核项。
- 章级/场景级审核、参考文风文件提取、单项重生成、双栏版本比较、审核批注和旧版恢复。
- 手动、自动和可暂停倒计时续写；重大冲突三选一、轻微冲突自动校正标记。
- Windows Credential Manager 密钥保存、项目级费用/预算、托盘后台继续和异常生成任务恢复状态。

### Fixed

- 正文生成现在由后端强制等待所有当前规划项批准，不能从前端绕过。
- 章节规划标题解析不再把 Agent 名称误识别为章节。
- 1024 像素宽度自动折叠侧栏，审批按钮在请求期间禁用，避免重复提交。

## [1.0.0] - 2026-07-18

### Added

- 小说项目、写作工作台、资料库、版本、回收站和中文示例数据。
- 五种模型协议、可编辑 Provider 预设、自定义 JSON/SSE/NDJSON/Chunked/Raw Adapter。
- 能力探测与降级、Route、限流、预算、熔断、成本和调用账本。
- 可版本化 Agent、React Flow DAG、并行调度、SSE、取消、重试和派生运行。
- 可解释上下文检索、FTS、Pin、分类、Token 预算和 Provider 数据边界。
- 人工审批、Diff、状态提取、逐项变更、冲突解决和事务写回。
- 完整备份/恢复、九种导出、诊断包、日志管理和恢复状态页。
- Windows 安装包、卸载器、免安装 ZIP、PyInstaller 运行时和 SHA-256 清单。

### Security

- 本地 Host/Origin 限制、安全 Header、SSRF 与 DNS rebinding 防护、凭据隔离、Secret 脱敏、受限模板和备份安全校验。
- 恢复与写回失败均使用事务回滚，生产桌面包关闭开发 API 文档。

### Performance

- Workflow 历史游标分页、流式 rAF 批处理、React Flow 可见元素渲染和实体状态批量查询。
- 加入 100 节点 DAG、10 万字符章节和 1,000 实体检索回归。

### Fixed

- 桌面启动器不再把 Edge 的首个转交进程退出误判为应用关闭；后台现在跟随真实 Novel Agent Studio 窗口生命周期。
- 安装器会等待卸载器的临时清理进程真正结束，避免卸载后立即重装时快捷方式或卸载注册表项发生竞态。
